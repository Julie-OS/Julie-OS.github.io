Question-Réponse

Comment mettre à jour Julie ?

Pour la version 3.0 : Il vous suffit de télécharger la nouvelle version (3.1 ou superieur) et de remplacer l'ancienne version par celle-ci.

Pour la version 3.1 : La mise à jour peut se faire automatiquement au démarrage de Julie (connexion à Internet nécessaire), ou directement comme pour la version 3.0.

Comment accéder à Internet sur Julie ?

L'accès à Internet à partir de Julie se fait par l'intermediaire du logiciel "Internet" disponible dans la section "Outils", ce logiciel utilise "Powershell" et il est donc important de disposer de cette application sur Windows pour se rendre sur Internet. Attention, comme nos serveurs JulieOS ne disposent pas d'une securité optimale il est fortement deconseillé d'utiliser cette application avec une connexion Internet de type "réseau Wifi public" c'est-à-dire les réseaux Wifi non soumis à un mot de passe disponible dans certains lieu tel que McDonald.

Mes données personelles seront-elles utilisées ?

Nous accordons à nos utilisateurs une grande transparance vis-à-vis de l'utilisation de vos données, à partir de la version 3.1 certaines de vos données pourront-être utilisées avec votre accord afin de connaître les possibles problèmes que rencontrent les utilisateurs (erreurs aux moments de démmarages d'aplications, applications que vous utilisez le plus, temps d'utilisation...), elles seront anonymisées avec un code correspondant à chacun des utilisateurs, ce code à 5 caractères représente les initiales de l'utilisateur, son genre et son jour de naissance. Par exemple un utilisateur nommé Julien Pitel né le 15/01/1990 aura comme code "JP15g". En cas de refus de communication de vos données personelles, nous limiterons la collecte de données à certaines erreurs de Julie en ne recuperant que le jour de l'erreur et pas l'heure précise. Dans tout les cas ces données ne seront jamais diffusées ou vendues à des fins publicitaires.

Qu'est-ce que le "Pack Son" et comment l'utiliser ?

Lors du téléchargement de Julie vous pouvez choisir de télécharger Julie Pack Son, c'est un dossier qui contient de la musique et des bruitages afin d'avoir du son sur Julie. Lors de la configuration de Julie vous pourrez l'installer, si Julie est déjà installée et que le Pack Son n'est pas enregistré vous n'avez qu'à le télécharger, le mettre dans le même dossier que JulieOS et dans JulieOS aller dans "Paramètres" puis "Systèmes Externes" et cliquer sur "Rechercher un pack son".

Quelles-sont les endroits cliquables sur Julie ?

Julie s'utilise majoritairement avec la souris, sur ce logiciel, vous pouvez cliquer sur tout les cadres avec une seule ligne comme celui-ci:

   ┌────┐ 

   │   ICI    │

   └────┘

JulieOS est un programme gratuit, aucun frais ne vous seront demandés au moment du téléchargement ainsi que des mises à jours.
